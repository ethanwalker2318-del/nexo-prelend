# TrustEx Prelend - Analytics Setup

## Как запустить аналитику

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск сервера
```bash
npm start
```

Сервер запустится на `http://localhost:3000`

### 3. Просмотр аналитики

Есть два способа:

#### Способ 1: Dashboard в браузере (реал-тайм обновление)
Откройте в браузере: `http://localhost:3000/dashboard.html`

Здесь вы видите:
- 📊 Количество просмотров страницы
- 🔘 Количество кликов по кнопке Telegram
- 📝 Список последних событий в реал-тайме

#### Способ 2: Текстовый файл
Данные автоматически сохраняются в файл `analytics.json` в этой же папке.
Его можно открыть любым текстовым редактором.

### 4. Как это работает

- Когда кто-то открывает `index.html`, отправляется событие `page_view`
- Когда кто-то нажимает кнопку "Открыть в Telegram", отправляется событие `button_click`
- Все события логируются на сервере и сохраняются в `analytics.json`
- Dashboard обновляет данные каждую секунду

## Структура analytics.json

```json
{
  "pageViews": 10,
  "buttonClicks": 3,
  "events": [
    {
      "type": "page_view",
      "timestamp": "2026-04-22T10:30:45.123Z"
    },
    {
      "type": "button_click",
      "timestamp": "2026-04-22T10:30:50.456Z"
    }
  ]
}
```

## Остановка сервера

Нажмите `Ctrl+C` в терминале
